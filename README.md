# Snapchat UI/UX Design
A Snapchat-inspired mobile UI/UX prototype created using Figma.

## Features
- Login Screen
- Chat Interface
- Stories Section
- Explore/Lenses Page
- Add Friends Screen

## Tools Used
- Figma

## Objective
The objective of this project was to improve UI/UX design skills and create a clean and user-friendly mobile interface.
